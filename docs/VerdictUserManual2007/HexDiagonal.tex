\section{Diagonal}

This metric is the ratio of the minimum diagonal length to the maximum diagonal length:
\[
q = \frac{D_{\min}}{ D_{\max}}.
\]
Note that if $D_{\max} < DBL\_MIN$, we set $q = DBL\_MAX$.
